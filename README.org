* Build And Service Run

Build and validate the file-based engine:

#+begin_src bash
make -C src clean compile run
make -C src clean compile diff
#+end_src

Build the HTTP service container:

#+begin_src bash
docker build -t mademanifest-phase-4:latest src
#+end_src
