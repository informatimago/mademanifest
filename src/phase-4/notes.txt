**Notes**
- The Docker image is Linux-based with Go `1.22.0`.
- It clones this repository into `/workspace/poc_v2_bundle`.
- It builds and installs Swiss Ephemeris during image build.
- `install_name_tool` is a no-op stub inside the image to avoid macOS-only build steps failing on Linux.

**Build command**
```bash
docker build -f env/docker/Dockerfile.txt -t mademanifest-poc-v2 .
```

If you want a different image name, set `IMAGE_NAME` when running:
```bash
IMAGE_NAME=your-name ./env/docker/build.sh
IMAGE_NAME=your-name ./env/docker/run_poc.sh
```
