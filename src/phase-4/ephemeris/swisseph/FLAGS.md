Swiss Ephemeris runtime rules for PoC V2

- Swiss Ephemeris version: 2.10.03
- Source archive: swisseph-2.10.03.zip (GitHub release)
- Ephemeris data path: ephemeris/data/REQUIRED_EPHEMERIS_FILES

Required flags and modes:
- Tropical zodiac only
- No sidereal modes
- No ayanamsha
- No topocentric corrections
- No heliocentric mode
- No nutation suppression
- Default Swiss Ephemeris precision
- Use ephemeris data files only from the provided data directory

Node policy:
- Mean Node for astrology outputs
- True Node for Human Design outputs

Any deviation from these rules constitutes an invalid implementation for PoC V2.
